# taiwan_travel

## DEMO
https://ellentyyu.github.io/F2E_taiwan_travel/

## 功能
* 台灣旅遊景點搜尋
* 各景點介紹
* 台灣餐廳搜尋
* 台灣旅宿搜尋
 
## 技術
* 交通部TDX資料介接
* Vue.js
* Composition API
* Vue CLI
* Vue Router
* Bootstrap 5
* SCSS

## 安裝與運行
```
# 安裝
npm install

# 開發伺服器
npm run serve

# 編譯(dist)
npm run build
```
